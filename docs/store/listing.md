# Play Store 등록 문구 (SpotLink)

## 앱 이름 (30자 이내)
SpotLink — 인터넷 없는 메신저

## 짧은 설명 (80자 이내)
인터넷 없이 근처 기기끼리 BLE 메시로 연결되는 오프라인 메신저. 종단간 암호화로 안전하게.

## 자세한 설명 (4000자 이내)

SpotLink는 인터넷·기지국·와이파이 공유기가 전혀 없어도 동작하는 오프라인 메신저입니다. 주변 기기들이 블루투스(BLE)로 서로 연결되어 하나의 메시 네트워크를 이루고, 텍스트와 파일이 기기에서 기기로 직접 전달됩니다.

산·바다·해외여행, 사람이 몰려 데이터가 안 터지는 축제·경기장, 재난으로 통신이 끊긴 상황에서도 곁에 있는 사람들과 계속 연락할 수 있습니다.

주요 기능

■ 인터넷 없이 동작하는 BLE 메시
모든 기기가 송신자이자 중계자가 되는 대칭 메시 네트워크. 별도 서버나 설정 없이 앱을 켜면 주변 기기와 자동으로 연결됩니다.

■ 멀티홉 중계
직접 연결이 안 닿는 상대에게도 중간 기기들이 릴레이해 전달합니다 (최대 64홉).

■ 만나면 전달되는 store-and-forward
수신자가 지금 근처에 없어도 메시지를 보관했다가, 나중에 어떤 경로로든 만나면 자동으로 전달합니다. 텍스트는 앱을 재시작해도 유지됩니다.

■ 종단간 암호화
X25519 + AES-256-GCM으로 수신자만 읽을 수 있게 암호화됩니다. 중계하는 기기도 내용을 볼 수 없습니다. QR 코드로 상대의 공개키를 직접 확인해 중간자 공격을 차단합니다.

■ 전달 영수증과 자동 정리
수신자가 받으면 서명된 영수증이 네트워크에 퍼져, 사본을 보관하던 중계 기기들이 자동으로 정리됩니다.

■ 파일 전송
사진·문서를 청크 분할 + 무결성 검증(SHA-256)으로 안전하게 전송. 같은 와이파이에 있으면 고속 전송으로 자동 전환됩니다.

■ 백그라운드 수신
화면이 꺼져 있어도 포그라운드 서비스가 메시 노드를 유지해 메시지·파일 도착을 알림으로 알려줍니다.

개인정보

SpotLink는 서버가 없고 어떤 데이터도 수집하지 않습니다. 회원가입·전화번호·광고·분석 SDK가 전혀 없으며, 모든 데이터는 사용자 기기에만 저장됩니다.

블루투스 권한은 근처 기기 탐색·연결에만 사용되며 위치 파악에 사용되지 않습니다(neverForLocation).
